# Amazon Clone (HTML + CSS)

A responsive Amazon homepage clone built using pure HTML and CSS.

This project recreates the Amazon landing page UI including:

* Responsive Navbar
* Search Bar
* Hero Section
* Product Grid Layout
* Product Cards
* Footer Section
* Hover Effects
* Responsive Design for Mobile Devices

## 🚀 Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Media Queries

## 📸 Features

* Clean Amazon-style UI
* Responsive layout for different screen sizes
* Hover animations
* Organized product sections
* Footer with multiple columns
* Beginner-friendly frontend project
